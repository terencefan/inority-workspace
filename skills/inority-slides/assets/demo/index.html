Template 02 / Signal Dashboard

Signal

先建立气场，再讲信息。

品牌型 slides 的第一目标不是列点，而是先建立观众对语气、调性、秩序和野心的感知。

Signal intensity
01 Tempo

滚动速度、停顿和动画节拍共同定义“品牌说话方式”。

02 Contrast

大的留白、硬边框和高对比色块让每一屏像一个镜头切换。

03 Depth

前景、粒子、模糊、视差，让页面更像空间而不是平面。

Template 03 / System Conveyor

System

它不是一个 deck，而是一套展示系统。

Cover Story Proof Closure

Cover

大字标题 + 运动背景，负责第一印象。

Story

用滚动推进 narrative，让“翻页”变成镜头语言。

Proof

图表、数据、案例以更编排化的方式进场，而不是直接平铺。

Closure

最后一屏收束价值主张，而不是普通结束页。

Template 04 / Split Reveal

Contrast

用一屏把 “Before / After” 讲透。

这类页面适合方案升级、品牌重构、体验改版、旧世界到新秩序的切换叙事。

Before Flat, crowded, literal

信息密度高但没有镜头感，观众会把它当普通网页。

After Layered, branded, directed

结构更少，但节奏、对比和空间感更强，像演示不是文档。

Template 05 / Spotlight Grid

Focus

让观众一次只看一个重点。

Spotlight 适合讲能力矩阵、产品卖点、模块组合。每个卡片都可以单独被点亮。

Focus 01 Statement

先用一句强话术定调，而不是立刻列需求。

Focus 02 Proof

用指标、案例、视觉证据把“为什么”钉牢。

Focus 03 Sequence

一个卡一个卡点亮，比同时塞满更有控制感。

Focus 04 Closure

最后一张卡负责收束价值，不让页面停在信息堆里。

Template 07 / Metric Cascade

Proof

让数字按节奏落下来。

适合讲增长、转化、覆盖、效率。重点不是“放数字”，而是让数字成为被编排的镜头对象。

Recall uplift 0

品牌记忆提升可以用强节奏的数字落版去锚定。

Engagement 0

倍率类指标适合配合小数滚动和短暂发光收束。

Launch speed 0

效率类数字可以和动线、轨道、刻度标尺组合讲。

Template 08 / Guided Timeline

Sequence

分阶段推进，而不是一次铺满路线图。

适合讲 roadmap、执行分期、品牌 campaign 节奏。每一步都像被镜头点亮，而不是整张图一次给完。

Phase 01 Frame the story

先确定观众要被带进什么情绪，再决定资料怎么出现。

Phase 02 Reveal the system

把模块、能力、关系按顺序组织，而不是做成拼贴板。

Phase 03 Close with force

最后一屏负责收束价值主张，让离场时还能留下记忆。
