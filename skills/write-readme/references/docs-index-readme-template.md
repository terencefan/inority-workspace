# 文档索引 README 模板

适用于 docs 目录首页、专题索引页或阅读导航页。

```md
# <主题名> 文档说明

> 用一句话说明这组文档覆盖什么主题，适合什么读者阅读。

## 说明

说明这组文档的边界、用途和阅读收益。

## 目录结构

| 路径 | 说明 |
|------|------|
| `foo/` | `<目录说明>` |
| `bar.md` | `<文档说明>` |

## 阅读顺序

1. `<先读什么>`
2. `<再读什么>`
3. `<需要时补充阅读什么>`

## 相关文档

- [文档名称](./example.md)
- [外层索引](../README.md)
```
